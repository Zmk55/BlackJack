### **Windows Users**

### **Linux Users**
```bash

sudo dpkg -i blackjack-ssh-client-1.0.0.deb
sudo systemctl start blackjack

wget https://github.com/Zmk55/BlackJack/releases/download/v1.0.0/blackjack-ssh-client-1.0.0.rpm

sudo systemctl start blackjack

```
